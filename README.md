# VerrixAI

AI-powered legal document analysis. Upload a contract, lease, NDA, or any legal document — get a plain-English breakdown in under 30 seconds, with risk flags, key obligations, and a "what this actually means" rewrite.

**Live site:** [verrixai.com](https://verrixai.com)

## What it does

Drop in a PDF, DOCX, or TXT file (or paste text) and you get back:

- A clear summary of what the document is
- Risk flags graded HIGH / MEDIUM / LOW, each with a short explanation
- The key obligations and clauses worth knowing about
- A plain-language rewrite — no legalese

Document text is extracted in your browser. The original file never reaches our servers. Only the analysis result is saved to your account.

## Tech stack

Static HTML / CSS / JS on Vercel · Supabase (auth + database) · Anthropic Claude (analysis) · Stripe (subscriptions) · Resend (transactional email).
